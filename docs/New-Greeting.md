# New-Greeting

## SYNOPSIS
This command let's you create a wonderful greeting to whoever you want!

## SYNTAX

```
New-Greeting [-Name] <String> [[-GreetingText] <String>] [<CommonParameters>]
```

## DESCRIPTION
This command let's you create a wonderful greeting to whoever you want!

## EXAMPLES

### Example 1
```
PS C:\> New-Greeting -Name "Sergei"

Hello Sergei
```

Will greet Sergei!

### Example 2
```
PS C:\> "Sergei" | New-Greeting

Hello Sergei
```

Will greet Sergei!
